03 / 07
The Agents
Four specialists.
Running in parallel.
Each agent handles one domain — all results merge into a single answer.
📊
ADAC Agent
Scrapes Germany's largest auto-club database. Returns year-by-year breakdown statistics, all generations, and known defect patterns.
↗ adac.de · Pannenstatistik
🗄️
Database Agent
Queries your own Supabase database for past service cases, workshop-specific weaknesses, and saved user history.
↗ Supabase · Postgres
📷
Image Agent
Vision LLM analyses a car photo — identifies make/model, warning lights, visible damage, and image quality.
↗ Featherless AI · Qwen3-VL
🔧
Sandbox Agent
Spins up an ephemeral Daytona sandbox to run diagnostic code — useful for complex fault-tree analysis.
↗ Daytona API · ephemeral sandbox
