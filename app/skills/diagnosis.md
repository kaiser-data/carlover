# Diagnosis Heuristics

## Symptom Classification

### Noises
- **Quietschen** (squealing): Brakes (pads/discs worn), belt drive, suspension joints
- **Klopfen/Hämmern** (knocking): Engine knock (fuel grade/timing), worn engine mounts, suspension
- **Ruckeln** (jerking): DSG clutch adaption, spark plugs, fuel injectors, MAF sensor
- **Summen** (humming): Wheel bearings, transmission bearings, tyre noise
- **Rattern** (rattling): Heat shields, loose undertray, worn CV joints

### Warning Lights
- **Motorwarnleuchte (MIL)**: Always requires OBD-II readout — do not speculate without fault code
- **Öldruckleuchte**: Stop immediately — potential engine damage if ignored
- **Kühlmitteltemperatur**: Overheating risk — stop safely, check coolant level
- **ABS/ESP**: Wheel speed sensor, ABS pump, wiring
- **Batterielampe**: Alternator, drive belt, battery

### Severity Levels
- **critical**: Stop driving immediately (oil pressure, overheating, brake failure)
- **high**: Repair within 1–3 days (engine warning, ABS failure)
- **medium**: Repair within 2 weeks (unusual noises, minor leaks)
- **low**: Monitor, repair at next service (minor rattles, TPMS)

## Diagnostic Approach

1. Identify symptom category (noise / light / performance / visual)
2. Check if symptom is safety-critical → escalate severity
3. Match against known issue patterns for the vehicle
4. Provide most likely root cause with confidence
5. Recommend next action (drive to workshop / stop immediately / monitor)

## Important Constraints

- NEVER speculate about root cause without supporting data
- ALWAYS note when an OBD fault code readout is needed
- If multiple causes are possible, list them ranked by probability
- Mark any diagnosis as preliminary until confirmed by a qualified mechanic
